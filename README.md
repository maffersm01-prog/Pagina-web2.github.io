# Cafetería Jackie Shop  
Proyecto de página web básica con HTML y GitHub Pages

## Introducción  
Este proyecto consiste en la creación de una página web sencilla utilizando **HTML semántico**, con el objetivo de practicar la estructura básica de una página, el uso de encabezados, párrafos, imágenes, enlaces, listas y formularios.  
Además, se implementó control de versiones con **Git, se creó un repositorio en GitHub, y finalmente el sitio fue publicado en línea utilizando GitHub Pages.


### Enlaces   
Se agregaron:
- Enlaces internos a las secciones: Nosotros, Promociones, Menú y Contacto  
- Enlaces externos a redes sociales (Facebook, Instagram, TikTok y YouTube)

### Imágenes
Se incluyeron imágenes del menú y de la receta, cada una con atributo `alt`.

### Listas   
El proyecto contiene:
- Lista del menú de navegación  
- Lista de promociones  
- Lista ordenada con los pasos de la receta  

### Formulario básico 
Dentro de la sección de contacto se agregó un formulario con:
- Input de nombre  
- Input de correo  
- Textarea (mensaje)  
- Botón para enviar  

### Control de versiones con Git  
Durante el desarrollo del proyecto se utilizaron los comandos:
- `git add .`
- `git commit -m "mensaje"`
- `git push`

###  Repositorio en GitHub  
El proyecto está alojado en un repositorio público.

###  Publicación con GitHub Pages  
El sitio se desplegó con GitHub Pages y es accesible de manera pública.



---

## ✅ Conclusión  
Este proyecto permitió poner en práctica los fundamentos de HTML, comprender el uso de estructuras semánticas, mejorar la organización del contenido, y aprender a usar Git y GitHub para llevar control de versiones y publicar proyectos web.  
El resultado es una página sencilla pero funcional que demuestra el uso correcto de los elementos solicitados.
